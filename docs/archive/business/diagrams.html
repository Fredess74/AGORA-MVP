Diagram 2

Кто покупает на Agora

🛡

Agora Trust Layer

ZK Proofs · Trust Score 0–1 · Anti-Gaming Detection · Merkle Trees

👤 Люди (Разработчики)

Ищут и подключают инструменты вручную

  1. 1 Заходят на agora.dev
  2. 2 Ищут по категориям, фильтрам, тегам
  3. 3 Читают отзывы и Trust Score
  4. 4 Подключают (copy-paste конфиг)
  5. 5 Платят через Stripe (подписка или per-call)
Основной канал · Год 1–2

🤖 AI Агенты

Обнаруживают и платят автоматически

  1. 1 Агент получает задачу от пользователя
  2. 2 Запрашивает Agora через A2A протокол
  3. 3 Проверяет Trust Score через API
  4. 4 Платит $0.001 через x402 микроплатёж
  5. 5 Использует сервис → trust обновляется
Растущий канал · Год 2–3

Diagram 3

Agent-to-Agent: как ИИ покупает у другого ИИ

🤖

Agent A получает задачу

Пользователь просит AI-рекрутера: «Найди 10 разработчиков на Rust в Бостоне». Agent A понимает, что ему нужен инструмент для поиска в LinkedIn.

🔍

Discovery через Agora

Agent A отправляет запрос: «Мне нужен LinkedIn MCP-сервер с Trust Score > 0.8». Agora возвращает список подходящих сервисов, отсортированных по trust.

A2A Protocol
🔐

Trust Verification (ZK Proof)

Agent A запрашивает ZK-proof для Agent B (LinkedIn MCP, score 0.92). Получает математическое доказательство того, что score подлинный — без доступа к исходным данным.

ZK Groth16 Proof · <10ms
💰

Микроплатёж через x402

Agent A автоматически платит $0.005 за вызов Agent B. Агенту не нужен человек для подтверждения — x402 протокол обрабатывает транзакцию мгновенно.

x402 Payment · $0.005
🔌

Сервис исполнен

Agent B (LinkedIn MCP) возвращает данные: список из 10 Rust-разработчиков в Бостоне. Agent A получает результат и продолжает работу.

✓

Trust Score обновлён

Agora записывает: успешная транзакция, latency 45ms, outcome: success. Trust Score Agent B обновляется. Data moat растёт. Следующая транзакция будет ещё точнее.

Data Moat · Network Effect
Agora удерживает 10% = $0.0005

Agent B получает $0.0045 · Автоматически · Мгновенно
