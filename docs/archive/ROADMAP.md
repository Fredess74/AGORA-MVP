## 🏆 COMPETITIVE LANDSCAPE

| Competitor | What they do | Our advantage |
|------------|--------------|---------------|
| **Zauth (zauthx402.com)** | x402 endpoint trust + crypto token ($ZAUTH) | ZK proofs, <10ms vs 500-1100ms, multi-protocol, SaaS model |
| **Masumi Network** | Agent payment layer | Trust + discovery + payments bundle |
| **MCP Marketplaces** | MCP server directories | Trust differentiation + payments |
| **Auth0 / WorkOS** | Agent identity | Full platform: identity + trust + marketplace |
| **Blockchain reputation** | Slow, expensive | Fast, cheap, ZK |
| **Internal agent ratings** | Siloed | Universal, portable |

> ⚠️ **См. [COMPETITIVE_ANALYSIS_ZAUTH.md](strategic/COMPETITIVE_ANALYSIS_ZAUTH.md)** — полный анализ главного конкурента

**Moat**:

1. **Marketplace lock-in** (creators list → consumers discover → network effects)
2. **ZK proofs** (cryptographic trust, not just endpoint uptime)
3. **Multi-protocol** (MCP + A2A + AP2 + x402 — шире всех конкурентов)
4. **Performance** (<10ms vs 500ms+ у конкурентов)
5. **Data advantage** (interaction history = accuracy)
