| **Owner** | Payments Team |

---

## What is x402?

Coinbase's HTTP 402 micropayment protocol. **Agora adds trust verification to payment headers.**

---

## Revenue Model

```
API request → 402 Required with agora_min_trust → Client pays → Agora earns 0.3%
```
